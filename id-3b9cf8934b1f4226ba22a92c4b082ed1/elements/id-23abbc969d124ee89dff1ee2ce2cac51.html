Clearances ()
The Online Clearance Request System is a secure Internet application that will allow users to search potential sub-contractors and request authorized clearance status documents. Clearance status documents describe if the contractors are in good standing with WCB. If contractors are not registered or in good standing with WCB, then the user of the system runs the risk of paying the assessment on their earnings as though the contractors are the users own employees. And if they have an accident on the job, user could be the one paying for it, with higher WCB assessment rates. If you become a registered user, you can also use this tool to compile and store lists of the sub-contractors you use most often and be updated on their clearance status via e-mail.
